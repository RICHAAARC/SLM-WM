# RivaGAN 外部 baseline

## 目录职责

- `source/`: 官方源码本地快照, 由 `external_baseline/.gitignore` 排除。
- 本目录只登记补充表候选 baseline 的来源边界。

## 登记范围

该 baseline 不进入主表命令计划，也不进入 `probe_paper`、`pilot_paper` 或 `full_paper` 的登记主张集合。若后续纳入统一对比，需要先冻结威胁模型并补充项目维护 adapter、三组负观测 fixed-FPR、公平命令计划、正式 observation、manifest、攻击矩阵记录和受治理导入校验器。
